## 简介
本项目是ASRT语音识别系统的Golang SDK项目，并实现了一个调用样例。

## Go Module 下载安装
```shell
$ go get github.com/nl8590687/asrt-sdk-go
```
## 软件环境要求
* 操作系统: 支持Windows 和 Linux、MacOS
* 开发环境： Go 1.16 + (未在更低的版本中做过兼容测试)
